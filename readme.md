# Recurso Ener API

Se trata de una API de estilo REST que utiliza la autenticación serializada JSON y OAuth2.

## Códigos de respuesta

* `200` OK
* `201` Created
* `204` No Content
* `401` Unauthorized
* `403` Forbidden
* `404` Not Found
* `422` Unprocessable Entity
* `5xx` Resource Guru is having trouble
